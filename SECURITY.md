# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in gh-social, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, please email: **ajang@duck.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 1 week
- **Fix or mitigation:** depends on severity, but we aim to resolve critical issues within 2 weeks

## Scope

This security policy applies to the latest released version of gh-social available on [PyPI](https://pypi.org/project/gh-social/).

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.2.x   | Yes       |
| < 1.2   | No        |
